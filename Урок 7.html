Блоки и семантика

Тэги бывают блочными и линейными. Первые занимают всю строку. Например, заголовки и параграфы. Линейные же идут друг за другом в одной строке. Для отображения границы блока можно использовать style=""

Lorem ipsum dolor sit, amet consectetur adipisicing elit. Voluptates facilis incidunt pariatur nihil non porro, necessitatibus tempora, ullam quas illum nam. Pariatur sit asperiores ratione velit labore. Assumenda, perspiciatis accusantium?

Для создания блоков используют 'div' с указанием id для навигации. Id в HTML должен быть уникальным. Id и Class могут быть описаны в блоках. Второй используется для стилистики и может быть группирован. Строчным аналогом 'div' является 'span'

Для быстрого создания 'div' с class и id можно использовать 'div.' и 'div#' соответственно
Вместо 'div' можно использовать 'header', 'nav', 'article', 'section', 'footer', 'aside'. Все они выполняют ту же функцию, что и 'div', но улучшают семантику
